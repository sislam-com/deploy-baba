# W-WEB: React/Vite SPA
**Path:** `web/` | **Status:** DONE (SEO prerender deferred to W-WEB.5 P3)
**Coverage floor:** smoke (≥1 render test per route) | **Depends on:** W-APIO, W-UI | **Depended on by:** W-CI (SPA deploy)

## W-WEB.1 Purpose

Replace all 15 Askama server-rendered templates with a Vite 6 + React 18 + TypeScript strict SPA (ADR-019). The SPA communicates with `services/ui` exclusively via JSON API (`/api/*`). Tailwind CSS moves from CDN script tag to PostCSS build plugin. The Lambda becomes a pure JSON API server + static asset host.

## W-WEB.3 Implementation Notes

### Stack

- Vite 6 — build tool (HMR, content-hash output, `base: '/'`).
- React 18 — UI.
- React Router 6 — client-side routing.
- TypeScript strict — `"strict": true`, no `any`.
- Tailwind CSS 3 via `postcss` + `tailwindcss` plugin (purges unused classes).
- `react-helmet-async` — per-route `<title>` + meta description.
- `openapi-fetch` + `openapi-typescript` — typed HTTP client; `just web-types` regenerates types from running `/api/openapi.json`.
- Vitest + Testing Library — unit + smoke tests (jsdom).

### Auth gate

`web/src/hooks/useAuth.ts` — calls `GET /api/auth/me` on mount. If `{ authenticated: false }`, redirects to `/auth/login`. All `/dashboard/*` routes wrap in `<RequireAuth>`.

### Dashboard dark theme (ADR-013)

Ported from Askama Tailwind CDN classes to Tailwind PostCSS config tokens. `tailwind.config.ts` adds the dark-palette:
- `dashboard-bg: '#0f172a'`, `dashboard-surface: '#1e293b'`, `dashboard-border: '#334155'`, `dashboard-text: '#e2e8f0'`, `dashboard-muted: '#94a3b8'`, `dashboard-accent: '#6366f1'`.
All `/dashboard/*` routes use `bg-dashboard-bg` root class.

### /ask chat UI

`routes/Ask.tsx` replaces the 200-line vanilla JS renderer in `templates/ask.html`:
- `react-markdown` + `remark-gfm` — markdown rendering (code blocks, tables, task lists).
- Custom `CitationBadge` component — renders `[source N]` references as clickable badges expanding to citation `url` + `excerpt` from `AskCitation[]`.
- Loading state + rate-limit error toast.

### New public JSON endpoints (W-UI additions)

Added in Phase D.1 — see ADR-019 for full list. Structs defined in `crates/api-openapi/src/models/` per ADR-012.

### Dev proxy

`vite.config.ts` proxies `/api → http://localhost:3000` in dev mode, so `just dev-stack` (Vite :5173 + Lambda :3000) works transparently.

### Build output

`pnpm --dir web run build` → `web/dist/index.html` + `web/dist/assets/*` (content-hashed). `index.html` gets `Cache-Control: no-cache`; assets get `public,max-age=31536000,immutable`.
