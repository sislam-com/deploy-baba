## Remaining Open Items

### CLI naming mismatch [FIXED ‚Äî W-XT.4.1]

Verification of `cargo xtask test all` revealed 5 mismatches (not 1 as originally documented).
Clap derives subcommand names via kebab-case from PascalCase variants, so `Format`‚Üí`format`,
`All`‚Üí`all`, and `Crate` is a subcommand (not `--crate` flag).

**3 justfile fixes applied:**
- `justfile:16`: `cargo xtask build fmt` ‚Üí `cargo xtask build format`
- `justfile:36`: `cargo xtask test --crate {{CRATE}}` ‚Üí `cargo xtask test crate {{CRATE}}`
- `justfile:48`: `cargo xtask quality gate` ‚Üí `cargo xtask quality all`

**2 quality.rs fixes applied:**
- `quality.rs:27,51`: `CoverageAction::Check { threshold: 80 }` ‚Üí `CoverageAction::Floors`
- `quality.rs:52+`: Added step 5 (cargo audit) to `run_all_gates()`

### `EnvironmentInterpolator` unused [DEFERRED ‚Äî W-XT.4.2]

**Location Correction**: EnvironmentInterpolator is a trait in `crates/config-core/src/lib.rs` (not xtask as originally documented).

**Decision**: Kept as intentional placeholder for future environment variable interpolation needs. Currently only used in doc examples and test code within config-core.
